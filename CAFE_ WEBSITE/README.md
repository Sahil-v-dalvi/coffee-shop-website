# Aura Coffee - Modern Landing Page

![Aura Coffee Preview](https://images.unsplash.com/photo-1497935586351-b67a49e012bf?auto=format&fit=crop&q=80&w=1200)

Aura Coffee is an ultra-premium, modern, and highly interactive landing page designed for a high-end specialty coffee shop. It features an immersive user experience built with HTML, CSS, and Vanilla JavaScript. 

This project goes beyond a standard template by introducing state-of-the-art web design aesthetics, including custom cursors, parallax effects, scroll-linked animations, and a full-screen fluid navigation system.

## ✨ Features
*   **Immersive Hero Section:** 3D subtle parallax background tied to mouse movement.
*   **Custom Interactive Cursor:** A unique geometric cursor that intelligently responds to hover states on interactive elements.
*   **Scroll-Driven Animations:** Elements elegantly fade and slide into view as you scroll down using the Intersection Observer API.
*   **Full-Screen Navigation:** A modern, mobile-friendly full-screen overlay menu with staggered text reveals.
*   **Deep Aesthetics:** Utilizes dark mode aesthetics, glassmorphism (frosted glass cards), and neon glow effects for extreme contrast.
*   **Fully Responsive:** Looks and functions perfectly on desktops, tablets, and mobile devices.
*   **Complete Website Sections:** Includes Hero, About Us, Menu Collection, Image Gallery, Customer Testimonials, Newsletter Subscription, Contact Form, and a rich Footer.
*   **Floating Action Button (FAB):** A persistent "Order Now" button for quick conversions.

## 🛠️ Technologies Used
*   **HTML5:** Semantic structure for optimal accessibility and SEO.
*   **CSS3:** Advanced usage of CSS Variables, Grid, Flexbox, Keyframe Animations, backdrop-filters (glassmorphism), and media queries.
*   **Vanilla JavaScript (ES6+):** For custom cursor logic, parallax effects, intersection observers, and DOM manipulation without relying on heavy libraries like jQuery.

## 🎨 Design Inspiration
The design takes inspiration from high-end luxury brands, utilizing a deep dark background (`#0d0c0c`) paired with warm gold/coffee accents (`#d4a373`). Typography pairs elegant serif headings (`Playfair Display`) with clean sans-serif body text (`Poppins`).
